## runs
ChatGPTに作ってもらったブラウザゲーム。
runs4.htmlが完成版です。（残りは製作途中でできたもので、利用しません。履歴のために残しているだけです。）
イメージは「自分のキャラを左右に操作しながら体力値を増やして、前から来る障害物を乗り越えていく」ようなゲームです。

+ 「自分」は画面の下中央におり、左右にのみ動けます。前後はできません。自分は「体力値」を持っており、後述の「パラメータ」や「敵」と接触することで増えたり減ったりします。
+ 「パラメータ（白）」は、画面上部から下部に降りてきます。パラメータは「四則演算」と「数値」からなり、たとえば「×3」や「-20」などで表現されます。「自分」が「パラメータ」を通過すると、自分の体力値がパラメータで表現された式で計算されます。たとえば体力値が「5」で、パラメータが「×3」であれば、パラメータ通過後の体力値は「5×3=15」になります。
+ 「障害物（赤）」も、画面上部から下部に降りてきます。障害物も数値を持っており、障害物と「自分」が接触した時、障害物の値が自分の体力値より大きければ、ゲームオーバーになります。
